# 🌍 FactVerse – Interactive Facts & Quiz Web App

> 🚀 Explore. Learn. Challenge Yourself.

FactVerse is a modern, interactive web application that allows users to discover **160+ amazing facts** across multiple categories and test their knowledge with a dynamic **quiz system**.

---

## ✨ Features

### 📚 Learn Mode
- Browse facts by categories:
  - 🌍 Earth  
  - 🚀 Space  
  - 🧠 Human Body  
  - 🦁 Animals  
  - 🏛️ History  
  - ⚗️ Science  
  - 💻 Technology  
  - 🍕 Food  
- Smooth animations & transitions  
- Random fact generator  
- Like ❤️ your favorite facts  
- Progress tracking  

---

### 🎯 Quiz Mode
- Multiple quiz categories  
- Custom number of questions (5–20)  
- Timer-based challenge ⏱️  
- Real-time scoring system ⭐  
- Instant feedback (correct/wrong)  
- Final performance analysis 📊  

---
